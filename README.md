# How to run Odoo

1. Download Docker\
https://www.docker.com/get-started

2. Create Docker Account\
https://store.docker.com

3. Install Docker
 
4. Open and Login Docker

5. Clone Project\
```git clone https://github.com/gapgag55/Odoo-with-Docker.git```

6. Go To Downloaded Project\
```cd Odoo-with-Docker```

7. Run odoo\
```docker-compose up ```

8. Open Web Browser\
```http://localhost:8069```
